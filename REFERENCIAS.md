## Librerías y Frameworks

### Core
- [React 19](https://react.dev/) - Biblioteca de UI
- [TypeScript](https://www.typescriptlang.org/) - Tipado estático
- [Vite](https://vitejs.dev/) - Build tool y dev server

### 3D
- [Three.js](https://threejs.org/) - Motor 3D para web
- [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber) - Renderer de React para Three.js
- [@react-three/drei](https://github.com/pmndrs/drei) - Helpers y abstracciones para R3F
- [@react-spring/three](https://www.react-spring.dev/) - Animaciones físicas para R3F

### Estilos
- [Tailwind CSS v4](https://tailwindcss.com/) - Framework CSS utility-first

### SEO
- [react-helmet-async](https://github.com/staylor/react-helmet-async) - Manejo de meta tags dinámicas

## Herramientas de Desarrollo

- [pnpm](https://pnpm.io/) - Package manager
- [ESLint](https://eslint.org/) - Linter para JavaScript/TypeScript
- [Vercel](https://vercel.com/) - Plataforma de despliegue

## Recursos de Diseño

- Paleta de colores: Deep Navy Blue (#0a192f, #112240) con acento en verde menta (#64ffda)
- Tipografía: Inter (sans-serif), JetBrains Mono (monospace)
- Inspiración: Portafolios modernos de desarrolladores con experiencias 3D interactivas
